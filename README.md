# SniX Cloud

Gestor de notas y snippets con nube: carpetas organizadas, sincronización
entre dispositivos, grupos compartidos con chat y un asistente integrado.

## 📥 Descarga

➡ **[Descargar la última versión](https://github.com/bryan1231996-rgb/snix-cloud/releases/latest)**

1. Descarga el `SniXCloud_Setup_*.exe` de la sección **Assets**.
2. Doble click e instala (no pide permisos de administrador).
3. Si Windows muestra el aviso azul de SmartScreen: pulsa
   **Más información → Ejecutar de todas formas** (es normal en apps
   nuevas sin firma digital).
4. Abre SniX Cloud, pulsa **Crear cuenta** y entra.

## ✨ Características

- 🗂 Notas y snippets de código organizados en carpetas, con resaltado de sintaxis
- ☁️ Sincronización en la nube entre tus dispositivos
- 👥 Grupos compartidos con notas colaborativas y chat integrado
- 🤖 Asistente SniX: busca, abre y crea notas por comandos (gratis, sin configurar nada)
- ✏️ Corrector ortográfico en español e inglés
- 🔒 Cifrado local de tu bóveda y bloqueo de pánico (Ctrl+L)
